Visual Basic - Speech Recognition Code:

*Windows Only

1. To edit open in Visula studio with .sln file
2. To run open the //speechstartexp/bin/Debug/speechstartexp.exe
3. Say Red, Blue or Green to change colour to use the voice recognition

4. Additional speech words can be added in switch case in the reco_SpeechRecognized function and declaration of speechList under the Sub Form1_load.
